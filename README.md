## :bulb: Setup

### Ruby setup

Run following command

```shell
bundle install
bin/rails db:create db:migrate db:seed
```

### Vue3 setup

Run following command

```
npm install
```

## :hammer: Run

### Start rails app

```shell
rails s
```

### Start vue app

```shell
npm run build
npm run dev
```
